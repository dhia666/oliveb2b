2. Run seed: `docker compose run --rm --user 0:0 --entrypoint wp wpcli oliveb2b seed --allow-root --path=/var/www/html`
3. Optional reset seed: add `--reset`
4. Visit: `http://localhost:8080/marketplace-search/`
5. Apply filters:
   - `Verified only`
   - `Supplier type`
   - `Near me radius` after clicking `Use my location`
