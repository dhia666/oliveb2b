## What Changed
- Added frontend inbox shortcode:
  - `[oliveb2b_my_inbox]`
- Inbox supports:
  - `Received` tab (messages where current user is recipient)
  - `Sent` tab (messages created by current user)
- Each entry displays:
  - subject
  - date
  - sender (for received)
  - delivery status (for sent)
  - related listing link
- Added inbox quick-link in logged-in header utility nav.
- Seeder now creates inbox page:
  - `/marketplace-inbox/`

## How To Test
1. Reseed:
   - `docker compose run --rm --user 0:0 --entrypoint wp wpcli oliveb2b seed --reset --allow-root --path=/var/www/html`
2. Visit:
   - `http://localhost:8080/marketplace-inbox/`
3. Send direct interaction messages from listing pages.
4. Confirm sent and received tabs show expected records.
