# T0.2 Notes

## What Changed
- Added Docker-based local WordPress environment with one-command startup.
- Added WP-CLI init to auto-install WordPress and set permalinks.
- Added a minimal GeneratePress child theme scaffold.
- Updated README with local run instructions and default admin credentials.
- Added `.env.example` and `.gitignore` for local configuration.

## How To Test
1. Run: `docker compose up -d`
2. Open: `http://localhost:8080` and confirm the site loads.
3. Open: `http://localhost:8080/wp-admin` and log in with the default admin.
4. In WP Admin, confirm permalinks are set to `/%postname%/`.
